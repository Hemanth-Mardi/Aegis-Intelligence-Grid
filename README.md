# Aegis Intelligence Grid 🚨

**Enterprise Offline-First Crisis Command Center**

A resilient full-stack platform that lets survivors send ultra-compact 17-byte SOS beacons (even completely offline) and delivers real-time AI-powered dashboards to NGOs, military, and satellite teams.

**Offline-First. Privacy-First. Action-First.**

## ✨ Highlights
- **Victim Beacon**: 17-byte binary packet, Dead-Reckoning, Wi-Fi fingerprinting, E2EE, P2P Mesh
- **Backend**: FastAPI + SQLite, AI Triage Engine, Real-time SSE
- **Dashboards**:
  - NGO: Incidents, triage & resource dispatch with Hope Loop ACK
  - Military: Tactical map + SAR HITL verification
  - Satellite: Hex inspector, decryption logs, live oscilloscope
- Strong security with spoof detection and low-bandwidth design

## 🛠 Tech Stack
- **Backend**: FastAPI, SQLite
- **Frontend**: Tailwind, Leaflet.js, Canvas
- **Simulators**: Python + JavaScript
